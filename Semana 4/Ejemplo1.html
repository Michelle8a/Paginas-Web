Introducción a CSS y sintaxis básica:
CSS, que significa "Cascading Style Sheets" (Hojas de Estilo en Cascada), es un lenguaje de diseño utilizado para estilizar la presentación de documentos HTML. Con CSS, puedes controlar el diseño, el formato y el aspecto visual de tu página web. La sintaxis básica de CSS implica seleccionar elementos HTML y aplicar estilos a través de reglas.
